Volcano Eruption Monitor v5.0

Real‑time Eruption Intelligence.

A dual-sensor monitoring platform for Arduino-based
volcanic activity detection — with AI command control.

Features
◎
Dual Sensor
Simultaneous ultrasonic distance and seismic ADC monitoring with independent alert thresholds.
⟁
Live Graphs
120-point rolling charts update in real time via Web Serial API at 500 ms intervals.
⌖
AI Control
Natural language commands via Sarvam AI. Describe actions in plain English — the AI executes them.
◻
Serial Shell
Full terminal with command history, tab-completion, and syntax validation before sending.
↓
Data Export
Auto-log every reading. Export as CSV or JSON with timestamps, sensor values, and alert flags.
◈
Alarm System
Audio beeps and visual flash overlays on threshold breach — dual alert escalates to 1.4 kHz tone.
